# Sous domaine 4 : Mécanique

| Expériences | Objectifs |
|---|---|
| 1- Lois de Newton | - Vérifier expérimentalement la deuxième loi de Newton. |
| 2- Chute verticale avec frottement. | - Mettre en évidence l'influence des frottements sur la chute verticale d'un solide dans des fluides. |
| 3- Mouvement d'un projectile dans un champ de pesanteur uniforme. | - Mettre en évidence les facteurs influençant la trajectoire d'un projectile. |
| 4- Relation quantitative entre la somme des moments et l'accélération angulaire. | - Vérifier expérimentalement la relation fondamentale de la dynamique dans le cas de la rotation autour d'un axe fixe. |
| 5- Système oscillant : (solide - ressort) | - Mettre en évidence les facteurs physiques influençant la période propre d'un oscillateur. |
| 6- Pendule de torsion. | - Étudier l'influence du moment d'inertie du pendule et de la constante de torsion du fil sur la période propre. |
| 7- Pendule pesant. | - Étudier l'influence du moment d'inertie du pendule sur la période propre dans les cas de petites oscillations. |
| 8- Résonance mécanique | - Étudier l'influence de la période de l'excitatore sur l'amplitude du résonateur. |
|  | - Étudier l'influence de l'amortissement sur la résonance. |
