Honest Lessons

Engineering Hurdles

Hallucination Risks

Small models tend to 'invent' math rules. RAG context injection solved 90% of accuracy issues.

Free-Tier Cold Starts

Services sleep on inactivity. Implemented loading states and backend warming to manage latency.

OCR Quality

Standard OCR failed on complex math formatting. Switched to Vision-based LLM extraction.

Memory Constraints

Running vector DBs in memory requires lean chunks. Optimized ChromaDB storage patterns.
